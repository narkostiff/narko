Автор содержимое страницы здесь с помощью GitHub ароматный Markdown, выберите шаблон созданный дизайнером, и публиковать. Идем дальше и продолжать макеты для их просмотра. Вы можете легко вернуться к отредактировать страницу перед публикацией. После публикации вашей страницы, вы можете вернуться к генератор страницы и переключиться на другую тему. В дополнение к поддержке регулярного содержания HTML, GitHub Страницы поддерживает Джекил, простой, блог осознавая статический сайт генератор, написанную собственной Том Престон-Вернер. Джекил позволяет легко создать заголовки веб-узле и нижние колонтитулы без необходимости копировать их через каждую страницу.
